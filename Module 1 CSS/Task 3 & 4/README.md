# Selectors Playground — Demo

Open `index.html` in your browser to view the demo. The demo shows:

- Universal selector `*` to reset margins and padding
- Element selector `h2` styling
- ID selector `#mainHeader` for the banner
- Class selector `.eventCard` for event containers
- Grouping selector `h3, p` to style headings and paragraphs together
- Color usage with HEX and RGBA
- Background image with a fallback color
- Gradient header styling using `background: linear-gradient(...)`
- Typography configured via Google Fonts

To preview quickly, right-click `index.html` and open with your browser or use a Live Server extension.
